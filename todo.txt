The project:
Enunciado do teste:
Criar uma listagem de fornecedores relacionado a uma empresa;
A empresa deverá ser composta por:
• Endereço;
• Nome Fantasia;
• CNPJ.
O fornecedor deverá ser composto por:
• Empresa;
• Nome;
• CPF ou CNPJ;
• Data/hora de cadastro;
• Telefone (Quantidade de telefones é variável).
Regras:
• O campo ‘Empresa’ em Fornecedor será um cadastro a parte;
• O campo ‘Endereço’ em Empresa deverá ser utilizado uma API para buscar o CEP
https://viacep.com.br/
• Caso a empresa seja do Paraná, não permitir cadastrar um fornecedor pessoa física menor de
idade;
• Caso o fornecedor seja pessoa física, também é necessário cadastrar o RG e a data de
nascimento;
• A listagem de fornecedores deverá conter filtros por Nome, CPF/CNPJ e data de cadastro.
Tecnologias:
• Backend:  Laravel
• Frontend: AdminLTE, Bootstrap 4.5

Tecchnical Table:
Framework: Laravel+Admin AdminLTE
System: This is a system that uses the Laravel framework with the AdminLTE UI to create a simple system of tracking Forcenedores and Empresas.
Regras de negócio:
O Menu principal contém os seguintes items:
Cadastro - To edit the login
Mudar Senha - Change the password
Listagem - This is the dashboard page, where the list os Fornecedores of an especific Empresa are listed. 
Cadastrar Empresa - Create an Empresa in the database
    Empresa as the following variables: 
    • Endereço;
    • Nome Fantasia;
    • CNPJ.
    • An array of Fornecedores 
Cadastrar Fornecedor - Create an Fornecedor in the database
    Fornecedor as the following variables: 
    • Empresa (the id of an company in the database that this Fornecedor is signed on);
    • Nome;
    • CPF ou CNPJ;
    • Data/hora de cadastro;
    • Telefone (An array of cellphones).
